# Python Automated Data Cleaning Tool

This project automates common data cleaning tasks for Excel and CSV datasets using Python.  
It is designed for data analysts who want a quick and repeatable way to clean raw data and generate professional reports.

---

## Step 1: Create Required Folders

Before running the script, you must create the following folders **with exact names**:

1. `input_data` – This folder will contain your raw data files (CSV or Excel).  
2. `output` – This folder will be automatically used to store cleaned files and reports.  
3. `output/charts` – This folder will store all charts generated by the script.

> Example folder structure:
> data-cleaning-project/
├── input_data/
├── output/
│ └── charts/
└── data_cleaning.py

---
## Step 2: Place Your Data File

- Place your **Excel (.xlsx)** or **CSV (.csv)** file inside the `input_data` folder.  
- Make sure the file name is **exactly correct** when prompted by the script.  

Example:
> input_data/
└── sales_data.xlsx

> ⚠ Important:  
> The script reads files **only** from the `input_data` folder.

---

## Step 3: Install Dependencies

Open terminal or command prompt and run:
pip install -r requirements.txt
The requirements.txt should include:
pandas
numpy
matplotlib
openpyxl

## Step 4: Run the Script

Run the script using:
python data_cleaning.py
The script will ask:
Enter file (Excel or CSV):
Type the exact file name you placed in input_data (e.g., sales_data.xlsx).

## Step 5: What the Script Does

Once executed, the script automatically:
Reads the input file
Fills missing numeric values using median
Removes duplicate rows
Detects and caps outliers using the IQR method
Saves cleaned data in output folder
Generates charts for missing values, outliers, distributions, and duplicates
Creates a structured Excel report documenting all steps

## Step 6: Output Files

After execution, the following outputs are generated:

1. Cleaned Dataset

Saved in the output folder:
CSV input:
output/cleaned_<original_filename>.csv
Excel input:
output/cleaned_<original_filename>.xlsx

2. Charts
Saved in output/charts/:
Missing values chart:
<filename>_missing_values.png
Outlier boxplot:
<filename>_outliers_boxplot.png
Distribution of the first numeric column:
<filename>_<column_name>_distribution.png
Duplicate records impact:
<filename>_duplicates_impact.png

3. Excel Report
Saved in output folder:
output/report.xlsx
Report contains:
Summary Stats – descriptive statistics of numeric columns
Missing Values – count of missing values per column (before cleaning)
Cleaning Log – rows before and after cleaning, duplicates removed

## Step 7: Final Result

After running the script, you get:
A cleaned dataset ready for analysis
Visual charts showing data quality
A structured Excel report documenting the cleaning process
